# JetBizCard

A practice project for learning **Jetpack Compose** with **Kotlin**.

This app renders a simple digital business card UI, built while following a Jetpack Compose tutorial and adapting it to a newer Android toolchain (AGP, Kotlin, and Compose Material3 library versions newer than the original tutorial).

## Tech stack

- Kotlin
- Jetpack Compose (Material3)
- Android Studio

## What it covers

- Composable functions (`Surface`, `Card`, `Column`, `Button`, etc.)
- Material3 theming (colors, typography, elevation)
- `Scaffold` with a `TopAppBar`
- Basic Compose state handling

See [`learning.md`](learning.md) for detailed notes and Q&A collected while building this project.
